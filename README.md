## 🤝 Contributing

We love your input! We want to make contributing to this Hyprland configuration as easy and transparent as possible. Please:

- 💡 First check if your idea has already been discussed in [Issues](https://github.com/pharmaracist/dots-hyprland/issues)
- 🐛 Report bugs using Github's [issue templates](https://github.com/pharmaracist/dots-hyprland/issues/new/choose)
- 📝 Make pull requests for bug fixes or new features
- 💬 Join our [Discord community](https://discord.gg/your-discord-link) for discussions

### 📜 Ground Rules
- Ensure cross-platform compatibility for every change you make
- Create issues for any major changes and enhancements
- Keep pull requests as small as possible
- Be welcoming to newcomers and encourage diverse new contributors

### 🔄 Pull Request Process
1. Fork the repo and create your branch from `main`
2. Update the documentation if needed
3. Make sure your code follows the existing style
4. Test your changes thoroughly
5. Issue the pull request!
